Ruby is...

a cross-platform, interpreted and object-oriented language. Yukihiro Matsumoto, commonly known as 'Matz' created the Ruby language in 1993. The year 2004 saw a massive surge of interest in Ruby, with the introduction of the Ruby on Rails Web application framework by David Heinemeier Hansson.

Ruby is a scripting language. There is no special main method in Ruby from which execution begins.

Parentheses are usually optional with a method call. These calls are all valid:


foobar

foobar()

foobar(a, b, c)

foobar a, b, c

Everything is an object

In Ruby, everything from an integer to a string is considered to be an object (more on this later). And each object has built in 'methods' (Ruby term for functions) which can be used to do various useful things.

exclude: Some methods such as puts and gets are available everywhere and don't need to be associated with a specific object.

Numbers in ruby

Numbers without decimal points are called integers, and numbers with decimal points are usually called floating-point numbers or, more simply, floats. 


          > 100.class
          => Fixnum
          
          > 10000000000000000000.class
          => Bignum
          
          > 100.0.class
          => Float
